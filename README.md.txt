# 池州悦跑俱乐部 - 跑步打卡网站

一个专为跑步团体设计的在线打卡和社交平台，支持跑步记录、照片分享、排行榜和活动组织。

## 功能特点

- 🏃‍♂️ 跑步打卡记录
- 📸 照片分享
- 📊 数据统计和排行榜
- 👥 跑友动态社区
- 📅 活动组织
- 📱 响应式设计，支持移动端
- 💾 本地数据存储

## 部署到 GitHub Pages

1. 在GitHub上创建新仓库
2. 将所有文件上传到仓库
3. 进入仓库设置 → Pages
4. 选择分支（通常是main）并保存
5. 等待几分钟，网站即可通过 `https://yuchuanjun22.github.io/joyrun` 访问

## 本地运行

直接在浏览器中打开 `index.html` 文件，或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server

# 使用 PHP
php -S localhost:8000
