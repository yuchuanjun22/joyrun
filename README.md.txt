# 池州悦跑俱乐部网站

池州悦跑俱乐部官方网站，为俱乐部成员和潜在会员提供信息和服务。

## 功能特点

- **响应式设计**：适配手机、平板和桌面设备
- **活动展示**：展示俱乐部各类跑步活动
- **会员注册**：在线提交会员申请
- **精彩瞬间**：展示俱乐部活动照片
- **LeanCloud集成**：使用LeanCloud作为后端数据存储

## 技术栈

- HTML5
- CSS3 (使用CSS变量和Flexbox/Grid布局)
- JavaScript (ES6+)
- LeanCloud (后端即服务)
